
# stock basic app

A stock app which performs basic task eg: Buy stock, sell stock. a simple demo prototype app which act like real stock brocker app



## Technolgies used

- python
- flask
- postgress
- docker
- nignex
- sql




## Installation 

cloan the reposirty

```bash
  git clone [project_link]
```

get into the working folder

```bash
  cd stock_basic_app 
```

Hope you have installed docker allredy in your system

run docker compose file

* windows
```bash
  docker compose up
```

* linux 
```bash
  sudo docker compose up
```
open url

```bash
  http://localhost:8080
```
